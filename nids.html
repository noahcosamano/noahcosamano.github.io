Network Intrusion Detection System

A Python-based IDS detecting port scans, abnormal traffic, and suspicious network activity, with a built-in interface to monitor filtered traffic in real-time.

Overview

This project demonstrates the design and implementation of a custom network intrusion detection system. It monitors live network traffic, identifies suspicious patterns, and provides real-time alerts for potential security threats.

Features

  • Detection of port scans and unusual traffic behavior.
  • Real-time traffic visualization interface.
  • Automated alerts and logging for suspicious activity.
  • Integration with Python scripts for extensibility.

Tech Stack

Python, Scapy, Tkinter (GUI), threading, queue management, network packet analysis.
